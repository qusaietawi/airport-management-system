# Airport Management System

## Description
This project is an airport management system that handles flights and passengers using different data structures.
It demonstrates the use of linked lists, stacks, queues, and cursor-based arrays in a real-world scenario.

## Features
- Manage flights and passenger data
- Use of linked lists (single and double)
- Stack and queue operations
- Cursor-based data structure implementation
- Simple graphical interface using JavaFX

## Technologies Used
- Java
- JavaFX
- Data Structures
